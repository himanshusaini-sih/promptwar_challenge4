# 📖 Overview

**StadiumAI** is an enterprise-grade **Generative AI-powered Smart Stadium Management Platform** designed to enhance the FIFA World Cup 2026 experience for fans, organizers, volunteers, staff, and administrators.

The platform leverages **Generative AI**, **real-time analytics**, **interactive navigation**, and **intelligent operational insights** to improve stadium operations, crowd management, accessibility, transportation, sustainability, and emergency response.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/StadiumAI.git

cd StadiumAI
```

---

## Frontend

```bash
cd frontend

npm install

npm run dev
```

Runs on:

```
http://localhost:5173
```

---

## Backend

```bash
cd backend

npm install

npm start
```

Runs on:

```
http://localhost:5000
```

---

# 🌱 Environment Variables

## Frontend (.env)

```env
VITE_API_BASE_URL=http://localhost:5000
```

---

## Backend (.env)

```env
MONGO_URI=mongodb://localhost:27017/stadiumai

GEMINI_API_KEY=YOUR_API_KEY

PORT=5000
```
